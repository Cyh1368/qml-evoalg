   - **Reintroduce parametrized controlled rotations** but use CRZ (instead of CRX) on center-connected edges, as CRZ provides better phase-space coverage for tic-tac-toe classification
   - **Maintain parameter efficiency** by using selective parameter sharing: corners share one RZ parameter, edges share another
   - This adds ~6-8 new parameters while maintaining circuit depth efficiency

4. **Rationale**:
   - RZ gates complement RY gates and the RX feature map, providing full SU(2) coverage per qubit
   - CRZ entanglers on center connections (1-8, 3-8, 5-8, 7-8) are geometrically meaningful for tic-tac-toe structure
   - Selective sharing keeps parameter count low while allowing the circuit to learn class-specific rotations
   - This design balances expressivity (to improve accuracy) with parameter efficiency (to maintain generalization)
